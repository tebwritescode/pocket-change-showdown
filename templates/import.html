CSV Format Requirements

Your CSV file should include the following columns:

  • Date - Format: YYYY-MM-DD or MM/DD/YYYY
  • Title - Name of the expense
  • Description - Detailed description (optional)
  • Category - Must match existing categories
  • Cost - Numeric value (e.g., 99.99)
  • Payment Method - Must match existing payment methods
  • Location - Where the expense occurred (optional)
  • Vendor - Vendor or merchant name (optional)
  • Notes - Additional notes (optional)
  • Tags - Comma-separated tags (optional)
Tip: Download our template to ensure your CSV is properly formatted.
